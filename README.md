# Android Camera App

This is an Android Camera Application developed using Java and Android Studio.
The app allows users to capture photos and view them in a separate screen.

## Features
- Capture photo using device camera
- Display captured image in Photo View screen
- Simple and clean UI

## Technologies Used
- Java
- Android Studio
- XML
- Git & GitHub

## How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the app on Emulator or Physical Device
